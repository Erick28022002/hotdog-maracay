Cómo configurarlo
3 pasos simples
Paso 1 — Crear Google Sheet
Ve a sheets.google.com → crea una hoja nueva → nómbrala "HotDogMaracay".

Paso 2 — Crear el Script
En la hoja: Extensiones → Apps Script → borra el código que hay → pega este código:
Haz clic en 💾 Guardar → luego en ▶️ Ejecutar (para dar permisos) → acepta todos los permisos.

Paso 3 — Publicar
Haz clic en Implementar → Nueva implementación → tipo: Aplicación web → "Quién tiene acceso": Cualquier persona → clic en Implementar → copia la URL y pégala arriba.
